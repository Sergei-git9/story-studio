# ИНСТРУКЦИЯ ДЛЯ ИИ: Создание JSON эпизодов Story Studio

## ЗАДАЧА
Создать JSON файл эпизода для Story Studio — веб-приложения чат-историй с эффектами.

## ФОРМАТ ФАЙЛА
```json
[
  { объект_сообщения_1 },
  { объект_сообщения_2 },
  ...
]
```

## ТИПЫ СООБЩЕНИЙ

### 1. Обычное сообщение (`msg`)
```json
{
  "type": "msg",
  "side": "left|right",
  "name": "Имя персонажа",
  "time": "21:00",
  "text": "Текст сообщения",
  "sound": "click"
}
```

### 2. Системное сообщение (`system`)
```json
{
  "type": "system",
  "text": "Описание сцены или события"
}
```

### 3. Системное с изображением
```json
{
  "type": "system",
  "text": "*Приходит фото*",
  "image": "stories/[story_id]/assets/images/photo.jpg",
  "sound": "camera"
}
```

### 4. Глитч-эффект
```json
{
  "type": "system",
  "text": "Текст с искажением",
  "glitch": true
}
```

## ОБЯЗАТЕЛЬНЫЕ ПОЛЯ
- `type` — всегда указывать
- `text` — всегда указывать

## ДОПОЛНИТЕЛЬНЫЕ ПОЛЯ
- `side` — "left" или "right" (только для msg)
- `name` — имя персонажа (только для msg)
- `time` — время в формате "ЧЧ:ММ"
- `sound` — звуковой файл
- `image` — путь к изображению
- `glitch` — true для эффекта искажения

## ДОСТУПНЫЕ ЗВУКИ
- `click` — обычные сообщения
- `system` — системные уведомления
- `glitch` — глитч-эффекты
- `camera` — звук камеры/фото
- `whisper` — шепот

## ПРАВИЛА НАПИСАНИЯ

1. **Время:** Увеличивать логично (21:00, 21:01, 21:02...)
2. **Стороны:** left = собеседник, right = главный герой
3. **Системные сообщения:** Описывать действия в *звездочках*
4. **Изображения:** Путь `stories/[story_id]/assets/images/filename.jpg`
5. **Глитч:** Использовать для драматических моментов
6. **Звуки:** Подбирать по смыслу сообщения

## ШАБЛОН ЭПИЗОДА
```json
[
  {
    "type": "system",
    "text": "Начальное описание сцены"
  },
  {
    "type": "msg",
    "side": "left",
    "name": "Персонаж1",
    "time": "21:00",
    "text": "Первое сообщение",
    "sound": "click"
  },
  {
    "type": "msg",
    "side": "right",
    "name": "Персонаж2",
    "time": "21:01",
    "text": "Ответ",
    "sound": "click"
  },
  {
    "type": "system",
    "text": "*Происходит что-то важное*",
    "sound": "system"
  },
  {
    "type": "system",
    "text": "*Приходит изображение*",
    "image": "stories/story_id/assets/images/photo.jpg",
    "sound": "camera"
  },
  {
    "type": "system",
    "text": "*Финальная драматическая сцена*",
    "glitch": true
  }
]
```

## ИНСТРУКЦИЯ ДЛЯ ВЫПОЛНЕНИЯ

**Когда пользователь просит создать эпизод:**

1. Спроси ID истории и номер эпизода
2. Узнай сюжет и персонажей
3. Создай JSON массив с сообщениями
4. Используй правильную структуру полей
5. Добавь эффекты и звуки по смыслу
6. Проверь валидность JSON
7. Укажи путь сохранения: `stories/[story_id]/ep[N].json`

**ВАЖНО:** Всегда создавай валидный JSON без синтаксических ошибок!
